Technical Architecture: The Multi-Modal Core

AEON is a sophisticated multi-modal AI, integrating specialized models to handle text, vision, and creative generation.

Small LLM

The Textual Brain

Processes, understands, and generates human-like text, optimized for local performance. Balances power with efficiency.

Model: Aeon 360M by Gustavo Kuklinski (GGUF)
Finetuned: Smollm2-360M by HuggingFace (Safetensor)

Text Embedding

The Semantic Engine

Converts text into numerical vectors to capture semantic meaning. Critical for accurate Retrieval-Augmented Generation (RAG) and web searches.

Model: Nomic Embed Text by Nomic AI (GGUF)

VLM (Vision-Language)

The System's Eyes

Allows the AI to "see" and interpret images. Links visual data with language to describe scenes and answer visual queries.

Model: SmolVLM 256M by HuggingFace (Safetensor)

Stable Diffusion

The Creative Artist

Generates high-quality images from text prompts, translating textual descriptions into unique visual art.

Model: Tiny-SD by Segmind (Safetensor)

Speak

The Voice of Aeon

Allow you to generate audio output from prompts or chat in voice.

Model: Piper TTS (ONNX)

Gemma3 270M

Plug another Brains

Use another LLM to generate context and knowloadge to Aeon.

Model: gemma3-270M-it (GGUF)

Source & Python Setup

Full control over your environment. Ensure you have requirements like **Python 3.11+**, **git**, and respective build tools installed.

# Clone with all plugins
$ git lfs install
$ git clone --recurse-submodules https://github.com/gustavokuklinski/aeon.ai.git

# Setup and Run
$ python3 -m venv .venv
$ source .venv/bin/activate
$ python install.py
$ python aeon.py
