The true power of AEON lies in its modular plugin system, which allows you to build a custom AI ecosystem. You can connect and switch between different LLM GGUF files for specific tasks or plug in alternative Stable Diffusion models for varied creative output. This modularity empowers you to create and integrate new features, tailoring AEON's capabilities to your exact needs and making it a truly evolutionary and agentic tool.

Default package

Speak

Function: Aeon's Voice
Type: text-audio
Capabilities: Transcribe text to audio files and allow audio chat
Source Code: Aeon Speak
Commands
/say [PROMPT]

VLM (Vision-Language Model)

Function: The system's "eyes".
Type: image-text
Capabilities: Allows the AI to "see" and interpret images. By linking visual data with language, it can describe scenes, answer visual queries, and identify objects within pictures also using a camera as "eyes".
Source Code: SmolVLM-256M-Instruct
Commands
/view [IMAGE_PATH] [PROMPT]

Stable Diffusion (Image Generator)

Function: The creative "artist".
Type: text-image
Capabilities: Generates high-quality images from text prompts. It translates textual descriptions into unique visual art, giving the system a powerful creative output.
Source Code: Tiny-SD
Commands
/image [PROMPT]

Gemma3 270m instruct

Function: Embed power.
Type: text-text
Capabilities: Embed Gemma3 270m instruct to work with Aeon default LLM
Model: Gemma3 270m Instruct
Commands
/gemma3 [PROMPT]
